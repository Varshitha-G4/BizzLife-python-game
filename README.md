# BizzLife

## A Python-based digital board game inspired by the popular board game "Business"

---

## Introduction

**BizzLife** is a turn-based board game built using **Python and Tkinter**.
Players move their pawns across a predefined board using dice rolls, manage virtual money through additions and deductions, and interact with different game rules as they progress.

The project focuses on:
- GUI-based game design using Tkinter
- Event-driven programming
- Separation of game logic and UI components

This project was developed as a learning exercise to understand GUI frameworks, game flow logic, and clean project structuring in Python.

## Features

- Dice-based pawn movement
- Visual board with fixed tile positions
- Banking system to add or deduct money per player
- Separate rule screen popup
- Keyboard-based pawn movement
- Modular code structure for easier extension

## How to Run the Project (User Instructions)

### Prerequisites
- Python **3.10+**
- Tkinter (comes pre-installed with Python on most systems)

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Varshitha-G4/BizzLife-python-game.git
   cd BizzLife-python-game
   ```
2. **Run the game**
    ```bash
    python BizzLife_Final.py
    ```

    - Instead of changing the background color, provide an option to change the color of individual pawns.
